# Data Dictionary

This document describes the columns in the synthetic datasets generated by `src/data/generate_synthetic_cases.py`.

## applications.csv

| Column | Type | Description |
|-------|------|-------------|
| `application_id` | string | Unique identifier for the application. |
| `customer_id` | string | Unique identifier for the customer. |
| `snapshot_date` | date (string) | Date of the application snapshot (YYYY‑MM‑DD). |
| `age` | integer | Age of the applicant. |
| `gender` | string | Gender of the applicant (M/F/None). |
| `marital_status` | string | Marital status (single/married/divorced/None). |
| `employment_status` | string | Employment status (employed/self‑employed/unemployed). |
| `sector` | string | Employment sector. |
| `years_in_job` | integer | Number of years in current job. |
| `monthly_income` | float | Monthly income. |
| `income_stability_score` | float | Score (0‑1) representing stability of income. |
| `existing_debt` | float | Total existing debt. |
| `debt_to_income_ratio` | float | Ratio of existing debt to income. |
| `requested_amount` | float | Amount requested for the new loan. |
| `requested_duration_months` | integer | Duration of the requested loan in months. |
| `product_type` | string | Type of loan product (personal/mortgage/auto/education). |
| `region` | string | Region or branch handling the application. |
| `account_tenure_months` | integer | Number of months the customer has held an account. |
| `prior_loans_count` | integer | Number of prior loans. |
| `prior_late_payments` | integer | Total number of late payments on prior loans. |
| `has_prior_default` | integer (0/1) | Indicator whether the applicant has ever defaulted. |
| `days_since_last_loan` | integer | Days since the applicant took their last loan. |
| `declared_purpose` | string | Purpose of the loan (home/car/business/education/holiday). |
| `free_text_note` | string | Unstructured note supplied by the applicant. |
| `monthly_installment` | float | Estimated monthly payment for the requested loan (requested_amount / requested_duration_months). |
| `other_monthly_debt_proxy` | float | Proxy for other monthly debt obligations (30 % of the monthly instalment). |
| `residual_income_proxy` | float | Estimated disposable income remaining after paying the new and other debt obligations. |
| `credit_score_proxy` | float | Synthetic credit score proxy (300–850) derived from late payment behaviour, debt‑to‑income, recency of borrowing, account tenure and employment stability. |
| `target_risk_flag` | integer (0/1) | Binary indicator of whether the applicant eventually defaulted (used for training only). |

## documents.csv

| Column | Type | Description |
|-------|------|-------------|
| `document_id` | string | Unique identifier for the document. |
| `application_id` | string | Identifier linking the document to the application. |
| `document_type` | string | Type of document (id_proof, income_proof, bank_statement, address_proof). |
| `is_required` | boolean | Indicates if the document is required. |
| `is_provided` | boolean | Indicates if the applicant provided the document. |
| `document_quality_score` | float | Quality of the OCR/extraction (0‑1). |
| `extracted_text` | string | Text extracted from the document (currently None for synthetic data). |
| `extraction_confidence` | float | Confidence in the extraction (0‑1). |

## reviews.csv

The reviews file is a placeholder reserved for recording human review information.  It contains columns for the review ID, application ID, predicted scores and classes, recommended action, the human decision, the reviewer’s risk class, their comments and the time taken to review.  Initially it is empty.
